## Preview

A dark, editorial-styled web page that displays code snippets with live filtering by category (Web, Backend, Scripting). Click any code block to copy it.

## Languages Used

| Language   | Category  | File         |
|------------|-----------|--------------|
| HTML       | Web       | index.html   |
| CSS        | Web       | style.css    |
| JavaScript | Web       | script.js    |
| Python     | Backend   | server.py    |
| Bash       | Scripting | —            |

## Getting Started

### Option 1 — Open directly
Just open `index.html` in your browser.

### Option 2 — Python server
```bash
python3 server.py
```
Then visit [http://localhost:8080](http://localhost:8080).

## Project  Structure

```
hello-world-polyglot/
├── index.html   # Page structure
```

## Features

- Filter cards by language category
- Click any code block to copy it to clipboard
- Fully responsive layout
- No dependencies — pure HTML/CSS/JS frontend
